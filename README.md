# tweet-bot
A bot that participates to twitter retweet contests.


# REQUIREMENTS
You need to create a `credentials.ts` file in src like this :
```
export const credentials = {
  consumer_key        : "...",
  consumer_secret     : "...",
  access_token        : "...",
  access_token_secret : "..."
};
```


# INSTALL
clone repo then :
- `npm i tsc -g`
- `npm i typings -g`
- `npm i`


# RUN
`npm start`
